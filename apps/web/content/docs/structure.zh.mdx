## 关键文件

* **`apps/web/src/middleware.ts`**：包含路由保护逻辑和重定向
* **`apps/web/src/routes.ts`**：定义核心路由以实现一致的导航
* **`packages/db/src/schema.ts`**：数据库 schema 定义
* **`apps/web/next.config.ts`**：Next.js 配置
* **`source.config.ts`**：Fumadocs 配置
* **`env.example`**：环境变量模板

## 代码组织最佳实践

在使用 AeloKit 时，请遵循这些代码组织最佳实践：

<Steps>
  <Step>
    ### 按功能分组，而不是按类型

    将相关文件保持在一起，无论其类型如何。
  </Step>

  <Step>
    ### 保持组件专注和小巧

    每个组件应该有单一职责。
  </Step>

  <Step>
    ### 巧妙使用统一导出

    创建 `index.ts` 文件从目录重新导出组件，使导入更简洁。

    ```tsx
    // components/ui/index.ts
    export * from './Button';
    export * from './Card';
    export * from './Input';

    // 使用
    import { Button, Card, Input } from '@/components/ui';
    ```
  </Step>

  <Step>
    ### 分离客户端和服务器组件

    在客户端组件顶部使用 `'use client'` 指令，并将服务器组件保持为默认。
  </Step>
</Steps>
