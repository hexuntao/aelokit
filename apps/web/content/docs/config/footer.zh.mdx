定义网站页脚中的链接，每个项目都是 [NestedMenuItem](/docs/config#nestedmenuitem)。

```tsx title="src/config/footer-config.tsx"
export function getFooterLinks(): NestedMenuItem[] {
  const t = useTranslations('Marketing.footer');

  return [
    {
      title: t('product.title'),
      items: [
        {
          title: t('product.items.features'),
          href: Routes.Features,
          external: false,
        },
        // 更多产品链接...
      ],
    },
    {
      title: t('resources.title'),
      items: [
        {
          title: t('resources.items.blog'),
          href: Routes.Blog,
          external: false,
        },
        // 更多资源链接...
      ],
    },
    // 更多页脚部分...
  ];
}
```

页脚链接按列组织，每列都有标题。每列都有一个标题和一个链接列表。这种结构允许按类别清晰地组织链接。
