Manage social media links displayed on the website, each item is a [MenuItem](/docs/config#menuitem).

```tsx title="src/config/social-config.tsx"
export function getSocialLinks(): MenuItem[] {
  const socialLinks: MenuItem[] = [];

  if (websiteConfig.metadata.social?.github) {
    socialLinks.push({
      title: 'GitHub',
      href: websiteConfig.metadata.social.github,
      icon: <GitHubIcon className="size-4 shrink-0" />,
    });
  }

  // Other social links based on configuration...

  return socialLinks;
}
```

It dynamically generates social media links based on the URLs provided in the `website.tsx` configuration, ensuring consistency between the configuration and what is displayed to users.
