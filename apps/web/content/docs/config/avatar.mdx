Define menu items in the user avatar dropdown menu, each item is a [MenuItem](/docs/config#menuitem).

```tsx title="src/config/avatar-config.tsx"
export function getAvatarLinks(): MenuItem[] {
  const t = useTranslations('Marketing.avatar');

  return [
    {
      title: t('dashboard'),
      href: Routes.Dashboard,
      icon: <LayoutDashboardIcon className="size-4 shrink-0" />,
    },
    {
      title: t('billing'),
      href: Routes.SettingsBilling,
      icon: <CreditCardIcon className="size-4 shrink-0" />,
    },
    {
      title: t('settings'),
      href: Routes.SettingsProfile,
      icon: <Settings2Icon className="size-4 shrink-0" />,
    },
  ];
}
```

Avatar configuration defines the dropdown menu that appears when users click their personal avatar. It typically includes the following links:

* Credits balance (enabled via `enableCredits` in `website.tsx`)
* User dashboard
* Account settings
* Billing/subscription management
* Sign out option (configured by default, no additional configuration needed)

This menu provides users quick access to specific features without cluttering the main navigation.
