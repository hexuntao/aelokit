## 组件库注册表

AeloKit 使用 `Components.json` 文件集中管理所有 UI 组件库的注册信息，方便组件的添加、升级和版本追踪。

### 添加新组件集合

你可以从这里找到所有的[Shadcn/ui Components Registry Directory](https://ui.shadcn.com/docs/directory)，然后添加其他UI库的组件。

## Shadcn/ui 组件

位于 `apps/web/src/components/ui` 目录中，都是 [Shadcn/ui](https://ui.shadcn.com/docs/components) 的基础组件。此目录包含作为应用程序构建块的基础组件。

<Callout type="warn">
  * 不要修改 `apps/web/src/components/ui` 目录中的组件。
  * 任何修改可能会在与模板同步代码库时导致冲突。
  * 这些组件定期与官方 shadcn/ui 发布版本同步。
  * `apps/web/src/components/ui` 目录被排除在 Biome 代码检查和格式化之外。
</Callout>

## 添加自定义组件

向项目添加新组件时：

1. **选择正确的位置**：将组件放在适当的业务模块目录中
2. **遵循命名约定**：为组件使用描述性的 PascalCase 名称
3. **保持一致性**：遵循现有的组件结构和模式

添加新组件的示例：

```tsx
// apps/web/src/components/dashboard/AnalyticsChart.tsx
import { Card } from "@/components/ui/card"
import { Chart } from "@/components/ui/chart"

export function AnalyticsChart() {
  return (
    <Card>
      <Chart data={...} />
    </Card>
  )
}
```
